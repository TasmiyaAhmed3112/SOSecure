# SOSecure 🛡️

🚨 SOS Guardian – Emergency Safety Android App

A smart Android safety application that helps users quickly send SOS alerts during emergencies using one-tap activation, voice commands, and automatic location sharing.

Designed to provide fast, reliable, and hands-free help when every second matters.

📌 Problem Statement

During emergencies, users often:

Panic and waste time unlocking phones

Cannot type or call manually

Fail to share their exact location

This app solves that by enabling:
✔ Instant SOS trigger
✔ Automatic location sharing
✔ Voice activation
✔ Quick alert delivery to trusted contacts

✨ Features
🚨 Core Features

One-tap SOS button

Sends emergency SMS to saved contacts

Shares live GPS location

Loud alert sound for attention

🎙 Smart Features

Voice activation (hands-free trigger)

Background emergency detection

Stores audio recordings (optional evidence)

Works even when app is minimized

💾 Data Handling

SQLite database for:

Emergency contacts

SOS history logs

Audio file paths

Secure local storage

🎨 UI/UX

Clean, minimal interface

Big emergency button for quick access

Easy contact management

Dark/Light theme friendly

🛠 Tech Stack
Component	Technology

IDE	Android Studio

Voice	SpeechRecognizer API
